# SSL.io

### Описание
Это учебный проект по Десктоп-разработке, который представляет собой игру, созданную с использованием библиотеки [SFML](https://www.sfml-dev.org/). Проект разрабатывается в рамках учебного курса по разработке игр.

**Группа:** ПС-21

### Особенности проекта
- **Платформа**: Десктоп (Windows, Linux)
- **Технологии**: C++ и SFML
- **Основной функционал**: Реализация основ игровой логики, работа с графикой и анимацией, управление вводом пользователя, обработка событий и отрисовка на экране.
